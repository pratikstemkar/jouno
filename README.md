# Jouno

Travel with friends and find new travel buddies. Have a social environment to discuss new places to explore and get travel insights from pros.

## How to Run?

1. Start backend Go server

```bash
go mod tidy
make docker-run
make run
```

2. Start frontend Next.js app

```bash
cd web
npm i
npm run dev
```
